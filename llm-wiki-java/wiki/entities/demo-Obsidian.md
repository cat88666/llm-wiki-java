# Obsidian

> 在文中被视作 LLM Wiki 的可视化工作界面，相当于“IDE”。

## 基本信息
- **类型**：工具
- **领域**：Markdown 知识管理、笔记系统
- **关联时间**：文中作为 LLM Wiki 工作流工具出现

## 核心贡献 / 特点
- 提供 Markdown 文件浏览、链接跳转和图谱视图。
- 能与 LLM Agent 并行使用，形成“LLM 写，用户看”的协作模式。

## 在知识库中的出现
文中并不把 Obsidian 当成知识管理终局，而是把它当成 LLM Wiki 的一个 IDE 级界面。
