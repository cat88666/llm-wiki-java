## **三、 精选项目经验**

### **1. 亿级流量游戏平台中台化建设（平台组组长）**

**项目描述：** 主导从项目组模式向平台化转型，为千万级用户提供高性能网关及通用逻辑服务。

- **自研网关：** 基于 **Netty 自研高性能网关**，扩展 Netty 3 支持批量解码，并结合后台配置实现流量平滑切换与优雅下线。
- **极致性能调优：** 优化 GC 参数结合对象池技术，实现高负载下 **0 Full GC**；通过拆分结算事务与并发异步提交，将 P99 延迟从 **800ms 降至 100ms**。
- **架构解耦：** 采用依赖倒置（DIP）模式设计游戏接口，支持通过配置动态加载不同游戏逻辑类，大幅提升业务迭代效率。
- **可靠性保障：** 引入 **RocksDB** 作为 MQ 和 MySQL 落地兜底，确保在极端网络环境下数据的最终一致性。

### **2. CoinsOTC 区块链场外交易平台（核心架构）**

**项目描述：** 基于微服务架构的 P2P 数字资产交易平台，实现资产流转的安全性与高性能。

- **核心贡献：**
    - **分布式锁优化：** 基于 Redisson 开发低代码分布式锁注解方案，有效解决多微服务环境下账户资金操作的并发一致性问题。
    - **性能压测与优化：** 针对交易高峰期采用 **Server层队列合并DB操作**，通过 100ms 的微延迟换取了 30% 以上的系统吞吐提升。
    - **链上集成：** 整合 bitcoinj 与 web3j，实现主流代币（BTC/ETH）的开户、充提及冷热钱包划转逻辑。

### **3. CloudStack 云计算平台网络架构改造**

- **关键突破：** 针对中国多运营商环境，重写 CloudStack 底层网络架构以支持**电信/移动/联通多线路优化**。
- **插件开发：** 适配并集成深信服等硬件设备，自研设备插件替代虚拟路由（VR）功能，实现高性能硬件防火墙、NAT 及负载均衡集成。

## **四、 架构经验总结**

### **分布式与高并发架构**

- **服务治理：** 深度配置 Dubbo 服务降级、重试及滑动升级；利用 **Gateway** 建立统一鉴权与白名单验证体系。
- **存储演进：** 使用 **Sharding-JDBC** 实现用户/交易中心的分库分表与读写分离。

- **设计模式应用：** 灵活运用构造器、状态、原型等设计模式构建高内聚、低耦合的代码结构。
- **冷热数据处理：** 基于 **MongoDB** 解决海量终端 ID 信息的高频更新与异步存储。
